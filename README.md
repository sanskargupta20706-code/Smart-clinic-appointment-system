# Smart Appointment System

This is a web-based Smart Appointment System built using Python Flask and MySQL.  
It allows users to register and book appointments digitally.

## Features
- User Registration
- Appointment Booking
- Automatic Queue Number Generation
- Data stored in MySQL

## Technologies Used
- Python (Flask)
- MySQL
- HTML, CSS

## How to Run
Run the project using:
python main.py

Then open:
http://127.0.0.1:5000/ui
